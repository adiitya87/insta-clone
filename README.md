# Instagram Clone
A fully functional social media application built using the **MERN Stack** (MongoDB, Express.js, React, Node.js). This application allows users to sign up, create posts with image URLs, like posts, and comment in real-time.

# Features

* **Authentication:** Secure Login & Signup (JWT-based).
* **Create Posts:** Users can share photos via URL with captions.
* **Home Feed:** View posts from all users.
* **Interactions:** Like and Comment on posts instantly.
* **Profile System:** View user profiles (posts count, followers).
* **Responsive UI:** Clean, Instagram-like interface using React & Lucide Icons.

---

## Tech Stack

### **Frontend**
* **React.js** (Vite)
* **React Router DOM** (Navigation)
* **Axios** (API Requests)
* **Lucide React** (Icons)
* **React Hot Toast** (Notifications)

### **Backend**
* **Node.js & Express.js** (Server)
* **MongoDB & Mongoose** (Database)
* **JWT (JsonWebToken)** (Authentication)
* **Bcrypt.js** (Password Hashing)
* **Cors** (Cross-Origin Resource Sharing)
